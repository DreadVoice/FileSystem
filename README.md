# File System Search Engine

A lightweight command-line search engine built in Java that indexes files from a local directory and performs fast keyword-based searches using inverted indexing and TF-IDF ranking.

## Features

* Recursive file system traversal
* Inverted index generation
* TF-IDF based search ranking
* Modular Java architecture
* Command-line query interface

## Tech Stack

* Java
* Java Collections Framework
* File I/O

## Getting Started

### Compile

```bash
javac *.java
```

### Run

```bash
java Main
```

## Example Commands

```bash
index /path/to/directory
search java collections
```

## Project Goal

This project was built to explore core information retrieval concepts such as indexing, ranking algorithms, and efficient document searching using Java.
